# To Get Started

## Frontend

### setup frontend

```

pnpm install && cp .env.example .env.local

```

### run local

```
pnpm dev
```

### build

```
pnpm build
```

## Smart Contract

### compile

```

npm run compile:hardhat

or
npm run compile:foundry

```

### test

```
npm run test:hardhat

or

npm run test:foundry
```
